## All Logic Gates Using Verilog HDL

### Project Title

**Implementation of Basic Logic Gates Using Gate-Level Modeling in Verilog HDL**

### Project Description

This project implements all fundamental digital logic gates using Verilog HDL gate primitives. The design includes NOT, AND, OR, NAND, NOR, XOR, and XNOR gates. Inputs are applied through a testbench, and the outputs are verified using simulation waveforms and RTL schematics in Xilinx Vivado.

### Features

* Gate-Level Modeling
* Verilog HDL Implementation
* RTL Schematic Generation
* Functional Simulation
* Verification of Truth Tables
* FPGA Design Fundamentals

### Logic Gates Implemented

1. NOT Gate
2. AND Gate
3. OR Gate
4. NAND Gate
5. NOR Gate
6. XOR Gate
7. XNOR Gate

### Inputs

* `a` : Input A
* `b` : Input B

### Outputs

* `not_out`
* `and_out`
* `or_out`
* `nand_out`
* `nor_out`
* `xor_out`
* `xnor_out`

### Software Used

* Verilog HDL
